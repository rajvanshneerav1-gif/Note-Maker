# 📰 The UPSC Digest — Streamlit App

AI-powered newspaper notes for UPSC Civil Services aspirants.

## Setup

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the app
```bash
streamlit run app.py
```

### 3. Add your API key
- Get your free Anthropic API key from: https://console.anthropic.com
- Paste it in the sidebar of the app when it opens

## Features
- 📋 Paste any article → get structured UPSC notes instantly
- 📅 Browse notes by date
- 🏷️ Browse notes by category (Polity, Economy, IR, etc.)
- ⚡ Prelims quick facts
- 📝 Probable Mains questions
- 🔑 Key terms
- Notes are saved locally in `notes.json`

## Categories Covered
Polity & Governance, Economy, International Relations, Environment & Ecology,
Science & Technology, Social Issues, Security & Defence, History & Culture,
Geography, Miscellaneous
